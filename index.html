Web Dizajn

U doba modernih tehnologija kada živimo u vremenu u kome se informacione tehnologije razvijaju i napreduju munjevitom brzinom, web sajt je postao svakodnevnica. Poseduju ga, firme, preduzeca, organizacije, javne ličnosti ... i mnogi drugi.

Sajt koji vam predstavljamo je izrađen u eksperimentalne svrhe, ali će dobro poslužiti i kao reklama pomoću koje ćemo vam predstaviti i ostale sajtove koje smo izradili.

Platrofme na kojima se baziraju naši sajtovi su: HTML5, CSS3, jQuery, PHP, Flash/ActionScript.

HTML se više ne koristi za vizuelne efekte, već samo za opis strukture i sadržaja web strane. Napušteno je raspoređivanje elemenata na stranici pomoću tabela - sada se izgled stranica podešava isključivo pomoću stilova, tj. CSS-a.

jQuery je mala, brza i bogata JavaScript biblioteka. Može da učini da se HTML dokumenti pomeraju, manipuliše i rukuje događajima, praćenim animacijama u mnoštvo savremenih web pregledača. Svojom kompatibilnošću i svestranošću, jQuery je promenio način na koji milioni ljudi na svetu pišu JavaScript.

PHP je jezik za pisanje skriptova koji rade na serveru, osmišljen za upotrebu na Web-u. Unutar HTML stranica može se ugraditi PHP kod koji se izvršava kada posetilac web lokacije zatraži stranicu. Web server tumači PHP kod ugrađen u stranicu i generiše HTML kod ili drugu vrstu izlaznih podataka koje posetilac stranice može da vidi.

Multimedija

Multimedija predstavlja istovremeni prenos, prikaz i korišćenje podataka sa više različitih memorijskih medija kao što su tekstualni podaci, grafika, nepokretne slike, pokretne slike i video, govorni i audio podaci. Istovremeno korišćenje više medija veoma je pogodno za čoveka, posebno kada su po sredi informacije koje se vizuelno menjaju.

Flash ima široku primenu u stvaranju zanimljivih aplikacija koje su obogaćene grafikom, animacijom, zvučnim i video zapisom. Koristi se za stvaranje interaktivnih web sajtova i digitalnih animacija, a u kombinaciji sa ActionScript-om može stvoriti sofisticirane interaktivne projekte.
